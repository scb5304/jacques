Help

Jacques is a Discord soundboard bot that can invited to any server. He can then be summoned into a voice channel by typing a recognizable command in a text channel. If the text channel is not named "commands," Jacques will delete the message shortly after to keep the server tidy.

Playing Sounds

Currently, users cannot add their own sound clips to Jacques. The list of available sounds is available here at jacquesbot.io along with play statistics.

!

A lone exclamation point in any text channel will trigger Jacques to play a random sound clip.

!soundname

Jacques will play the passed sound provided that it exists.

!help

Links to Jacques' site to display list of stored sounds.

!helptext

Spits out a raw text dump of all sounds stored in Jacques' database.

Streaming Youtube Videos

!stream

Summons Jacques to the user's current voice channel to play the audio of the passed youtube URL. The link can optionally include the time at which the stream should begin.

Example: !stream https://www.youtube.com/watch?v=jdB4ER_-mSY

!volume

Prints the current stream volume (from 0 to 100). If a volume is passed, then it will be adjusted accordingly. If Jacques is currently streaming audio, the volume will be adjusted real-time. Stream volume does not affect the volume of Jacques' sound clips.

Example: !volume (prints volume)

Example: !volume 50 (sets stream volume to 50%)

Leaving a Voice Channel

!cancel

If Jacques is currently in the same voice channel as you, this will cause him to disconnect immediately.
